## [1.0.0-alpha.5] - 2026-04-16

### Crop Images in Results

This release introduced comprehensive support for cropping images within results. It added backend endpoints, frontend thumbnails, expandable rows, and lightbox features to enhance image interaction.

- Added crop image endpoint and thumbnail URLs in results
- Added thumbnail column to results table and client-side sorting
- Added expandable rows and accordion in results page
- Added crop lightbox with backend semaphore and keyed table rows
- Added progressive reveal with crop highlight on source page
- Added per-entry highlight and clipped thumbnails in entry rows
- Added adaptive score-based prediction truncation

### Secure and Validate Output

Security improvements focused on sanitizing HTML output and fixing accessibility issues. Validation was added to ensure safe file path resolution.

- Sanitized HTML output and added focus trap to fix security findings
- Validated resolved paths against storage base to prevent path traversal
- Addressed low severity security findings in results handling

### API and Frontend Enhancements

The API and frontend were improved for better consistency and usability. Centralized API URLs and added crop metadata to campaign results.

- Centralized API URL configuration and fixed relative URL 404 errors
- Added crop metadata to campaign results endpoint
- Updated test environment to match base URL conventions

### Performance Improvements in Results Pagination

SQL-level pagination and grouping were implemented to optimize results retrieval and display.

- Added SQL-level pagination and GROUP BY for results queries

### Code and Service Refactoring

Refactoring efforts improved code readability and modularity, especially in prediction building and CSV export.

- Extracted PredictionBuilder from duplicated logic
- Wired CSV export to OCR text parser formatting
- Streamed CSV export using yield_per and extracted HTTP concerns
- Consolidated test fixtures and simplified service code
- Removed dead providers and fixed logging in monitoring components
