## What's New — April 2026

### See Petition Images Right in Your Results

When you review matched signatures, you can now see the original petition image directly in the results table. Click any image to view it full-size with the relevant entry highlighted on the page — no more switching back and forth between files.

### Faster, Smarter Results

The results table now sorts instantly without reloading. Low-confidence matches are automatically filtered out, so you see the most relevant results first. Large result sets load faster thanks to improved database queries.

### Bug Fixes

Fixed an issue where some pages wouldn't load correctly when accessed from different URLs. Crop images now display reliably with the correct metadata included.

### Security Note

We strengthened image handling to prevent unauthorized file access. All HTML output is now sanitized to keep your data safe.
